# Easy-Readzy

Easy-Readzy は「技術書を気軽に読んでアウトプット」をテーマにした、所有書籍の管理と学習に役立つPC向けブラウザアプリです。
持っている本を検索して、読んで何を得たいのかを登録し、好きなタイミングで文章によるアウトプットを作成して本に紐付けることができます！

### リンク: https://www.easyreadzy.com

![Easy-Readzy](/images/logo.png)

## こんなお悩みはありませんか？

- 積読を抱えており、所有している技術書を把握しきれていない
- 何のために手に入れたのか目的を忘れてしまった
- 読んだけど読むだけで終わり、知識が定着した実感がない

これらの問題を、「所有書籍の一覧化」「書籍の登録時、何のために買ったのか(得たい知識)、どこに保管しているのか(本がある場所)、入力を必須化」「目標に対しての文章形式のアウトプット」によって解決しよう！という狙いのアプリです。

# How to use

### 基本は`本を検索`→`本を登録`→`本の詳細に遷移`→`詳細画面からアウトプットを作成`の流れです。

1. 上部のバーから持っている本を検索して探す

2. 本にを読む上での目標と保管場所を入力し、登録

3. 書影をクリックすると詳細画面に遷移

4. 詳細画面ではアウトプットの作成ができる

![demo-movie](/images/demo.gif)

## 技術スタック

- バックエンド: Ruby on Rails
- フロントエンド: HTML/CSS, JavaScript
- データベース: MySQL
- インフラ: Render, AWS(Amazon Route 53 / Amazon S3)

## 機能一覧

- ユーザー機能
    - ゲストログイン/ログアウト(devise gem)
- 書籍の新規登録機能
    - 書籍の検索(Google Books API)
    - 書籍情報の取得・登録
    - 書籍と紐付く目標・目的の登録
    - 書籍と紐づく保管場所の登録
- アウトプットの作成機能
    - 書籍の詳細
    - 目標と紐づくアウトプットの作成/編集/削除
- 所有書籍の検索機能
    - タイトル/著者名などのキーワードによる検索
    - 保管場所による検索
